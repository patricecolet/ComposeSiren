# ComposeSiren

ComposeSiren is a vst3 plugin that synthesize sounds of siren made by the [MÃ©canique Vivante][1] company. The plugin allows to handle the seven-piece Siren Orchestra: a piccolo, two sopranos, two altos, a tenor and a bass.

The plugin file, called **ComposseSiren.vst3**, is located in the folder: *builds/MacOSX/*.
